Lightcast 2024 Economic Complexity Dashboard

Interactive visualization of NAICS 6-digit industry complexity analysis across U.S. geographies

About This Dashboard

This dashboard visualizes economic complexity metrics derived from the Lightcast 2024 dataset using the Daboin et al. methodology. It analyzes 947 industries (6-digit NAICS) across 5 geographic aggregation levels.
